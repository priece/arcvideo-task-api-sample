# totalmedia-task-api-sample
A sample code for Arcvideo Live Task API calling.

# Features
- Create task by JSON file.
- Start task by ID.
- Stop task by ID.
- Delete task by ID.
- Get task info by ID.

# Needs
- Python >= 3.8
- pip install requests

# Run
```
yum install python3
pip install requests
python3 create_task.py
```
